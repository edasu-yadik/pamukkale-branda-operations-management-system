# Pamukkale Branda Operations Management System

A web-based operations management system developed for Pamukkale Branda to manage installations, invoices, customer payments and scheduling workflows.

## Features

- Installation Tracking
- Invoice Management
- Payment Tracking
- Email Parsing
- Dashboard & Reporting
- Google Calendar Integration

## Technologies

### Frontend
- React
- Axios
- React Router

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL
